## [0.1.0] - 2025-11-07

### Added
- Initial release of Microsoft Entra ID Learning Program
- Comprehensive 5-day training curriculum covering:
  - Day 1: Foundations & Identity Fundamentals
  - Day 2: Authentication & Security
  - Day 3: Enterprise Integration & Hybrid Identity
  - Day 4: Governance & Advanced Features
  - Day 5: Production Readiness & Capstone
- Three learning tracks: Fast Track (2 days), Standard Track (3-4 days), Comprehensive Track (5 days)
- Complete documentation suite:
  - Main program overview (README.md)
  - Detailed 5-day schedule with minute-by-minute breakdown
  - Enterprise Architecture & Integration Guide with ASCII diagrams
  - Complete Resource Links with 100+ curated Microsoft resources
- Interactive visualizations and tools:
  - Schedule Timeline (labs/schedule-timeline.html) - Interactive progress tracking with checkboxes
  - Architecture Explorer (architecture.html) - Mermaid.js diagrams visualizing all 7 architecture layers
  - GitHub Pages landing page (index.html) - Professional learning portal
- GitHub Pages infrastructure:
  - Jekyll configuration (_config.yml)
  - Comprehensive deployment guide (GITHUB_PAGES_SETUP.md)
  - Mobile-responsive design across all pages
- Developer resources:
  - CLAUDE.md for AI assistant context
  - VISUALIZATIONS_README.md detailing all interactive tools
  - .gitignore for proper repository hygiene
- Progress tracking features:
  - LocalStorage persistence for learning progress
  - Export progress as JSON
  - Real-time statistics and completion percentage
- All content based on November 2025 Microsoft Entra releases
- Free hands-on labs using Azure free account and M365 Developer Program
- Production-ready best practices and Zero Trust implementation guidance
- Capstone project: Design complete enterprise identity solution
